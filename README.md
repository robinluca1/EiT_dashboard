# EiT_dashboard

Click on any of the rooms to show their data. Including:

- Temperature
- Humidity
- CO2 levels
- Occupation
- Energy consumption
- Noise level
- Light level

3D model controls:
Drag the left mouse button to pan the model into view
Use shift + left click to rotate the model
